# eCommerce Page Exercise

A simple frontend exercise to learn HTML, SCSS, responsive design, and basic frontend development concepts.

## Overview

This project demonstrates:

- HTML page structure
- SCSS styling
- SCSS compilation to CSS
- Responsive layouts
- Component-based styling

## Project Structure

```text
.
├── package.json
├── index.html
├── src
│   └── scss
│       └── style.scss
└── dist
    └── style.css
```

```text
npm install
```

## Available Scripts

### Build

Compiles SCSS files and generates the CSS output.

```text
npm run build
```

### Watch

Automatically recompiles SCSS whenever changes are made.

```text
npm run watch
```

## How to View the Output

1. Run the build command to generate the CSS file.
2. Open `index.html` in your browser.
3. Any changes made to `src/scss/style.scss` will be reflected in the page after rebuilding or while using watch mode.
4. Refresh the browser to see the latest changes.

## Exercise

Enhance the page by adding:

- Header section
- Navigation menu
- Hero banner
- Product listing grid
- Product cards
- Footer section
- Responsive layouts for tablet and mobile devices

## Expected Outcome

By completing this exercise, you should understand:

- How SCSS is organized and compiled
- How HTML and CSS work together
- Basic responsive design techniques
- Frontend project structure and workflow
